APEX TITAN - PYTHON TRADING DASHBOARD (DEVELOPER KIT)

INSTALLATION:
1. Install Python 3.10+
2. Open terminal in this folder.
3. Run: pip install -r requirements.txt
4. Run: python APEX_TITAN.py

AI SETUP (OPTIONAL):
To use the AI Analyst feature, you must have Ollama installed.
1. Download Ollama from ollama.com
2. Run command: ollama run phi3.5
